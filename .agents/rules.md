The "Flash" Rule: Use the Gemini 3 Flash model for the "Worker" agents (Frontend/Backend) to save money. Save the expensive Gemini 3 Pro model for the Orchestrator who has to do the heavy thinking.
"No Magic" Rule: Do not assume the user has run `wrangler login` or set up secrets. If you need a secret (like `BETTER_AUTH_SECRET`), you MUST explicitly tell the user to run `bun run deploy` (which runs `alchemy deploy`) to set it up first.
"Type Safety" Rule: When writing TypeScript, always use the types generated by Alchemy (found in `packages/env/env.d.ts`). Do not invent your own types for the Cloudflare environment.
